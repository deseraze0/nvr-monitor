# NVR Monitor

A modular Python platform for monitoring NVRs and IP cameras.

## Features

- Event monitoring
- Telegram notifications
- SQLite/PostgreSQL
- FastAPI
- Modular drivers
- Multi-camera support
- Multi-NVR support

## Planned Drivers

- Watchdog
- Xiongmai (XMEye)
- Hikvision
- Dahua
- ONVIF
- YOLO
